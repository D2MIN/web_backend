Подключение к серверу kubsu-dev

Делаем команду ping для проверки целостности и качества соединений в сетях на основе TCP/IP

Вызов команды -type=A для просмотра адресной записи

Вызов команды -type=MS для просмотра портового шлюза и приоритета
